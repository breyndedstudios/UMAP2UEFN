# UMAP2UEFN WEB
UMAP2UEFN WEB is a local, HTML based UMAP converter.

Instructions:
  1. Open the UMAP2UEFN WEB Git-Hub Page.
  2. Click "Upload UMAP/JSON/TXT File".
  3. Clcik "Convert to UE text format".
  4. Press "Copy" to copy the output to your clipboard or press "Download TXT" to download a text file containing the output.
  5. Paste the copied text into UE (or a UE client ex. UEFN).


Please report all bugs on Git-Hub.
