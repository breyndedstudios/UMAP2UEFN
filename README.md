# UMAP2UEFN
UMAP2UEFN is an online, HTML based UMAP converter.

Instructions:
1. Open the UMAP2UEFN Git-Hub page or HTML file in your web browser.
2. Click "Upload JSON File".
3. Clcik "Convert to UE text format".
4. Copy the output usign the "Copy" button or download it as a text (.txt) file.
5. Paste the copied text into UE (or a UE client ex. UEFN).

Please report all bugs on Git-Hub.
